# 图集打包
* 测试场景：Scenes->Test->SpriteAtlas
* 打开Sprite Atlas: Edit > Project Setting > Editor > Sprite Packer > Mode > Always Enabled
* SpriteAtlas和其关联的所有资源打成一个AssetBundle包
* 大尺寸图片不要关联SpriteAtlas（尺寸界限视具体项目而定）
